# Zzz

Zzz 是一款 macOS 极简菜单栏多任务倒计时工具，旨在通过原生优雅的界面，提醒你关注重要的时间节点。

[English Version](./README.en.md)

## 核心功能

- **多任务倒计时**：不再局限于就寝提醒，支持自定义多个倒计时点（如睡眠、运动、喝水等）。
- **Emoji 图标**：为每个任务设置 Emoji 图标，支持搜索选择，一眼识别当前任务。
- **原生体验**：基于 SwiftUI 与 AppKit 打造，完美融入系统毛玻璃视觉与交互。
- **智能跨天**：支持跨凌晨作息逻辑（以 4:30 AM 为分割点），完美契合熬夜或特殊作息。
- **动态 UI**：全新的卡片式交互界面，支持星期重复配置，操作舒展。
- **状态栏显示**：支持隐藏图标的紧凑模式，并能在临近目标时自动切换显示单位。
- **桌面组件**：基于 WidgetKit 实现，支持桌面实时查看多个计时任务。
- **视觉警报**：倒计时在临近目标或超时状态下会自动切换为红色。
- **快捷操作**：支持右键菜单一键退出。

## 安装方法

1. 从 Release 页面下载 Zzz.dmg 或自行从源码构建。
2. 打开 DMG 文件，将 Zzz 拖入 Applications 应用程序文件夹。
3. 启动应用，即可在菜单栏看到倒计时。

## 开发与构建

### 环境要求

- macOS 14.0 或更高版本
- Xcode 15.0 或更高版本
- XcodeGen

### 从源码构建

1. 克隆本仓库。
2. 安装 XcodeGen（如果尚未安装）：`brew install xcodegen`。
3. 在项目根目录执行：`xcodegen generate` 以生成 Xcode 工程。
4. 打开 `Zzz.xcodeproj`，选择 Zzz 方案并进行构建。
